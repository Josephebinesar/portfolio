Create a professional portfolio website for a software developer focused on job opportunities.
The website should feel clean, modern, human-designed, and recruiter-friendly, using light colors and elegant spacing.

The design should not look like a generic template. It should feel like a personal brand website for a developer and AI enthusiast.

Use soft light color palettes like white, soft blue, light purple, and pastel gradients.

The website must be fully responsive and optimized for desktop, tablet, and mobile.

Include smooth animations and an always-running subtle animated background across the entire site. The background should feel modern and tech-inspired, such as:

• slowly moving gradient waves
• floating particles
• subtle neural-network-style lines
• soft glowing blobs

The background animation should be smooth and not distracting so the content remains readable.
